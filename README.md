# E-Commerce Data Analysis

## Overview
This project performs exploratory data analysis on an e-commerce dataset using Python and Jupyter Notebook. The goal is to understand customer purchasing behavior, product trends, and sales patterns.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Objectives
- Analyze customer purchase behavior
- Identify popular product categories
- Study sales trends
- Visualize insights using graphs

- Identified trends in product purchases
- Visualized sales distribution
- Explored customer buying patterns
